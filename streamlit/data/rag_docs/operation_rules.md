## 사분면 분류 (`classify_quadrant()`)

X축 지표(기본값 cpa)와 Y축 지표(기본값 margin_rate) 각각의 평균을 기준으로 매체/그룹을 4분류한다.

```
          Y축(마진율) 높음
              │
   소재 개선  │  최우수
   (저효율)   │  (best)
──────────────┼────────────── X축(효율) 좋음
   비효율     │  단가 점검
   (loss)     │  (expensive)
              │
          Y축 낮음
```

- CPA처럼 "낮을수록 좋은" 지표는 X축 방향을 반전해서 판정한다 (`is_lower_better`).
- best: X축도 좋고 Y축도 좋음 → 최우수
- loss: X축도 나쁘고 Y축도 나쁨 → 비효율(손실)
- expensive: X축은 나쁘지만(비용 비효율) Y축은 좋음(마진 양호) → 소재 개선 후보
- low_eff: X축은 좋지만 Y축은 나쁨 → 단가 협업 대상

## CVR·마진율 알람 이모지 기준

| 조건 | 라벨 |
| --- | --- |
| clk < 30 또는 turn < 1 | 판단 보류 |
| CVR >= 평균 AND 마진율 >= 평균 | 최우수 |
| CVR < 평균 AND 마진율 >= 평균 | 소재 개선 후보 |
| CVR >= 평균 AND 마진율 < 평균 | 수익율 점검 필요 |
| CVR < 평균 AND 마진율 < 평균 | 표시 안 함 |

## 룰 기반 운영 알림 (`generate_alerts()`)

| 조건 | 레벨 | 메시지 패턴 |
| --- | --- | --- |
| margin_rate < 0 | red | "주의: {매체명} — 마진율 {n}% (비용 초과)" |
| cvr < 평균 cvr × 0.7 | yellow | "관찰: {매체명} — CVR {n}% (평균 대비 낮음)" |
| cvr >= 평균 AND margin_rate > 10 | green | "양호: {매체명} — CVR {n}% / 마진율 {n}% 안정" |

알림은 red → yellow → green 순으로 정렬된다.
